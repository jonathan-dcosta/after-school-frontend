This is the Vue.js frontend for the After-School Lessons coursework (CST3144).

## Features ##

1. Display list of lessons from the backend (via Fetch)
2. Search lessons by subject or location
3. Sort by subject, location, price, or availability
4. Add lessons to cart and view checkout page
5. Client-side validation for name and phone number
6. Connected to Express.js backend deployed on Render

## How to run locally ##

1. Open 'index.html' in a browser
2. Make sure the backend server is running and the URLs in 'main.js' point to it
